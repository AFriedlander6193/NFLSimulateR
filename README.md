## Installation

### NOTE:
Make sure you have Rtools45 installed

### Recommended (pak)

```r
install.packages("pak")
pak::pak("AFriedlander6193/NFLSimulator")
```

### Alternative (remotes)

```r
install.packages("remotes")
remotes::install_github("AFriedlander6193/NFLSimulator")
```
## Features

- Simulate NFL games play-by-play
- Generate realistic offensive and defensive personnel
- Create route combinations and coverage schemes
- Incorporate PFF player grades
- Bayesian models for play outcomes
